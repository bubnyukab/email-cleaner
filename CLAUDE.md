## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues (`github.com/bubnyukab/email-cleaner`). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical labels with default names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo — one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
